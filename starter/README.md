# Starter Folder

This directory contains the main notebook used for the SentimentScope capstone project.

## Primary File

- `SentimentScope_starter.ipynb`: the core notebook for data preparation, transformer modeling, training, validation, and test evaluation

## Local-Only Artifacts

Some files may exist locally during development or experimentation, but they are intentionally excluded from version control to keep the public repository lightweight:

- extracted IMDB dataset files
- downloaded dataset archive
- trained model checkpoints
- executed notebook copies
- notebook checkpoint folders

## Reproducing The Project

1. Install dependencies from the repository root with `pip install -r requirements.txt`.
2. Download the IMDB dataset locally.
3. Open `SentimentScope_starter.ipynb`.
4. Run the notebook end-to-end.
